# M TECH - Premium Hardware E-Commerce

A modern, responsive e-commerce platform for gaming and professional tech hardware. Built with vanilla JavaScript, HTML5, and CSS3.

## Features

- Dynamic product catalog with real-time API integration
- Smart search functionality to filter products
- Fully responsive design (mobile, tablet, desktop)
- User registration with email & password validation
- Service price calculator
- Cookie consent banner
- Scroll-to-top button

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **API**: Fake Store API (electronics)
- **Fonts**: Google Fonts (Orbitron, Exo 2)

## Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/m-tech.git
cd m-tech

# Open with a local server
python -m http.server 8000
# Visit: http://localhost:8000
```
